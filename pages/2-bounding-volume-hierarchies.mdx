# Bounding Volume Hierarchies

## The Key Idea

The key idea of creating bounding volumes for a set of primitives is to find a volume that fully encloses (bounds) all the objects. For example, suppose you computed a bounding sphere of 10 objects. Any ray that misses the bounding sphere definitely misses all ten objects inside. If the ray hits the bounding sphere, then it might hit one of the ten objects. So the bounding code is always of the form:

```
if (ray hits bounding object)
    return whether ray hits bounded objects
else
    return false
```

A key thing is that we are *not* dividing the screen or the volume space. We are dividing the list of objects. Each object is in exactly one bounding volume, but bounding volumes can overlap.

## Hierarchies of Bounding Volumes

To make things sub-linear we need to make the bounding volumes hierarchical. For example, if we divided a set of objects into two groups, red and blue, and used rectangular bounding volumes, we'd have:

```
if (hits purple)
    hit0 = hits blue enclosed objects
    hit1 = hits red enclosed objects
    if (hit0 or hit1)
        return true and info of closer hit
return false
```

The BVH forms a tree. The root is the bounding box for the whole scene. Each internal node is the bounding box of its children. Leaf nodes hold one or a few primitives. This turns a linear scan through every object into a traversal of a binary tree — typically logarithmic in the number of objects.

## Axis-Aligned Bounding Boxes (AABBs)

To get good performance we need a good division into subsets, and to traverse the BVH quickly we need intersection tests that are very fast. We use axis-aligned bounding rectangular parallelepipeds (axis-aligned bounding boxes, or AABBs) for this. All you need to know is that testing a ray against an AABB is fast, and that AABBs work well in practice.

The slab method is based on the observation that an n-dimensional AABB is just the intersection of n axis-aligned intervals, often called "slabs". An interval is just the points between two endpoints, e.g., $x$ such that $3 \le x \le 5$. In 2D, two intervals stacked make a 2D AABB (a rectangle). In 3D, three intervals make an AABB.

<Tabs>
  <Tab label="Rust">
    ```rust filename="aabb.rs | AABB definition"
    #[derive(Clone, Copy)]
    pub struct Aabb {
        pub x: Interval,
        pub y: Interval,
        pub z: Interval,
    }

    impl Aabb {
        pub fn new(x: Interval, y: Interval, z: Interval) -> Self {
            Self { x, y, z }
        }

        pub fn from_points(a: Point3, b: Point3) -> Self {
            Self {
                x: if a.x() <= b.x() { Interval::new(a.x(), b.x()) } else { Interval::new(b.x(), a.x()) },
                y: if a.y() <= b.y() { Interval::new(a.y(), b.y()) } else { Interval::new(b.y(), a.y()) },
                z: if a.z() <= b.z() { Interval::new(a.z(), b.z()) } else { Interval::new(b.z(), a.z()) },
            }
        }

        pub fn from_aabbs(box0: Aabb, box1: Aabb) -> Self {
            Self {
                x: Interval::from_intervals(box0.x, box1.x),
                y: Interval::from_intervals(box0.y, box1.y),
                z: Interval::from_intervals(box0.z, box1.z),
            }
        }

        pub fn axis_interval(&self, n: usize) -> Interval {
            match n {
                0 => self.x,
                1 => self.y,
                _ => self.z,
            }
        }
    }
    ```
  </Tab>

  <Tab label="C++">
    ```cpp filename="aabb.h | AABB class"
    class aabb {
      public:
        interval x, y, z;

        aabb() {}

        aabb(const interval& x, const interval& y, const interval& z)
          : x(x), y(y), z(z) {}

        aabb(const point3& a, const point3& b) {
            x = (a[0] <= b[0]) ? interval(a[0], b[0]) : interval(b[0], a[0]);
            y = (a[1] <= b[1]) ? interval(a[1], b[1]) : interval(b[1], a[1]);
            z = (a[2] <= b[2]) ? interval(a[2], b[2]) : interval(b[2], a[2]);
        }

        aabb(const aabb& box0, const aabb& box1) {
            x = interval(box0.x, box1.x);
            y = interval(box0.y, box1.y);
            z = interval(box0.z, box1.z);
        }

        const interval& axis_interval(int n) const {
            if (n == 1) return y;
            if (n == 2) return z;
            return x;
        }
    };
    ```

## Ray Intersection with an AABB

The slab method tests a ray against an AABB by intersecting it with three pairs of planes (one per axis) and checking if the resulting intervals overlap. If they all overlap, the ray hits the box.

<Tabs>
  <Tab label="Rust">
    ```rust filename="aabb.rs | Ray-AABB intersection"
    impl Aabb {
        pub fn hit(&self, r: &Ray, mut ray_t: Interval) -> bool {
            for axis in 0..3 {
                let ax = self.axis_interval(axis);
                let adinv = 1.0 / r.direction()[axis];
                let t0 = (ax.min - r.origin()[axis]) * adinv;
                let t1 = (ax.max - r.origin()[axis]) * adinv;
                if t0 < t1 {
                    if t0 > ray_t.min { ray_t.min = t0; }
                    if t1 < ray_t.max { ray_t.max = t1; }
                } else {
                    if t1 > ray_t.min { ray_t.min = t1; }
                    if t0 < ray_t.max { ray_t.max = t0; }
                }
                if ray_t.max <= ray_t.min {
                    return false;
                }
            }
            true
        }
    }
    ```
  </Tab>

  <Tab label="C++">
    ```cpp filename="aabb.h | hit function"
    bool hit(const ray& r, interval ray_t) const {
        for (int axis = 0; axis < 3; axis++) {
            const interval& ax = axis_interval(axis);
            const double adinv = 1.0 / r.direction()[axis];
            auto t0 = (ax.min - r.origin()[axis]) * adinv;
            auto t1 = (ax.max - r.origin()[axis]) * adinv;
            if (t0 < t1) {
                if (t0 > ray_t.min) ray_t.min = t0;
                if (t1 < ray_t.max) ray_t.max = t1;
            } else {
                if (t1 > ray_t.min) ray_t.min = t1;
                if (t0 < ray_t.max) ray_t.max = t0;
            }
            if (ray_t.max <= ray_t.min) return false;
        }
        return true;
    }
    ```

## Creating Bounding Boxes of Lists of Objects

A `HittableList`'s bounding box is the union of the bounding boxes of all its children, accumulated as objects are added.

## The BVH Node Class

The BVH node is itself a `Hittable`. It stores two children (also `Hittable`s, which may be primitives or other BVH nodes) and their combined AABB. Traversal checks the box first; only if the box is hit do we recurse into children.

<Tabs>
  <Tab label="Rust">
    ```rust filename="bvh.rs | BvhNode"
    pub struct BvhNode {
        left: Arc<dyn Hittable>,
        right: Arc<dyn Hittable>,
        bbox: Aabb,
    }

    impl Hittable for BvhNode {
        fn hit(&self, r: &Ray, ray_t: Interval, rec: &mut HitRecord) -> bool {
            if !self.bbox.hit(r, ray_t) {
                return false;
            }
            let hit_left = self.left.hit(r, ray_t, rec);
            let hit_right = self.right.hit(
                r,
                Interval::new(ray_t.min, if hit_left { rec.t } else { ray_t.max }),
                rec,
            );
            hit_left || hit_right
        }

        fn bounding_box(&self) -> Aabb {
            self.bbox
        }
    }
    ```
  </Tab>

  <Tab label="C++">
    ```cpp filename="bvh.h | bvh_node class"
    class bvh_node : public hittable {
      public:
        bvh_node(hittable_list list) : bvh_node(list.objects, 0, list.objects.size()) {}

        bool hit(const ray& r, interval ray_t, hit_record& rec) const override {
            if (!bbox.hit(r, ray_t)) return false;
            bool hit_left = left->hit(r, ray_t, rec);
            bool hit_right = right->hit(r, interval(ray_t.min, hit_left ? rec.t : ray_t.max), rec);
            return hit_left || hit_right;
        }

        aabb bounding_box() const override { return bbox; }

      private:
        shared_ptr<hittable> left;
        shared_ptr<hittable> right;
        aabb bbox;
    };
    ```
  </Tab>
</Tabs>

## Splitting BVH Volumes

To build the tree we recursively split the list of objects. The simplest strategy: pick a random axis, sort the primitives along it, and split the sorted list in half. The recursion bottoms out at one or two objects.

We also need a small utility to generate random integers:

<Tabs>
  <Tab label="Rust">
    ```rust filename="rtweekend.rs | random_int"
    pub fn random_int(min: i32, max: i32) -> i32 {
        random_double_range(min as f64, (max + 1) as f64) as i32
    }
    ```
  </Tab>

  <Tab label="C++">
    ```cpp filename="rtweekend.h | A function to return random integers in a range"
    inline int random_int(int min, int max) {
        // Returns a random integer in [min,max].
        return int(random_double(min, max+1));
    }
    ```
  </Tab>
</Tabs>

<Tabs>
  <Tab label="Rust">
    ```rust filename="bvh.rs | BvhNode::new"
    impl BvhNode {
        pub fn new(objects: &mut Vec<Arc<dyn Hittable>>, start: usize, end: usize) -> Self {
            let axis = random_int(0, 2) as usize;
            let object_span = end - start;
            let (left, right): (Arc<dyn Hittable>, Arc<dyn Hittable>) = if object_span == 1 {
                (objects[start].clone(), objects[start].clone())
            } else if object_span == 2 {
                let (a, b) = (objects[start].clone(), objects[start + 1].clone());
                if box_compare(&*a, &*b, axis) == Ordering::Less {
                    (a, b)
                } else {
                    (b, a)
                }
            } else {
                objects[start..end].sort_by(|a, b| box_compare(&**a, &**b, axis));
                let mid = start + object_span / 2;
                (
                    Arc::new(BvhNode::new(objects, start, mid)),
                    Arc::new(BvhNode::new(objects, mid, end)),
                )
            };
            let bbox = Aabb::from_aabbs(left.bounding_box(), right.bounding_box());
            Self { left, right, bbox }
        }
    }
    ```
  </Tab>

  <Tab label="C++">
    ```cpp filename="bvh.h | bvh_node constructor"
    bvh_node(std::vector<shared_ptr<hittable>>& objects, size_t start, size_t end) {
        int axis = random_int(0, 2);

        auto comparator = (axis == 0) ? box_x_compare
                        : (axis == 1) ? box_y_compare
                                      : box_z_compare;

        size_t object_span = end - start;

        if (object_span == 1) {
            left = right = objects[start];
        } else if (object_span == 2) {
            left  = objects[start];
            right = objects[start+1];
        } else {
            std::sort(objects.begin() + start, objects.begin() + end, comparator);
            auto mid = start + object_span/2;
            left  = make_shared<bvh_node>(objects, start, mid);
            right = make_shared<bvh_node>(objects, mid, end);
        }

        bbox = aabb(left->bounding_box(), right->bounding_box());
    }
    ```

## The Box Comparison Functions

The comparators sort hittables by the minimum of their bounding box interval along a given axis.

<Tabs>
  <Tab label="Rust">
    ```rust filename="bvh.rs | Box comparison functions"
    fn box_compare(a: &dyn Hittable, b: &dyn Hittable, axis: usize) -> Ordering {
        let a_interval = a.bounding_box().axis_interval(axis);
        let b_interval = b.bounding_box().axis_interval(axis);
        a_interval.min.partial_cmp(&b_interval.min).unwrap_or(Ordering::Equal)
    }
    ```
  </Tab>

  <Tab label="C++">
    ```cpp filename="bvh.h | BVH comparison functions"
    class bvh_node : public hittable {
      // ...
      private:
        static bool box_compare(
            const shared_ptr<hittable> a, const shared_ptr<hittable> b, int axis_index
        ) {
            auto a_axis_interval = a->bounding_box().axis_interval(axis_index);
            auto b_axis_interval = b->bounding_box().axis_interval(axis_index);
            return a_axis_interval.min < b_axis_interval.min;
        }

        static bool box_x_compare(const shared_ptr<hittable> a, const shared_ptr<hittable> b) {
            return box_compare(a, b, 0);
        }

        static bool box_y_compare(const shared_ptr<hittable> a, const shared_ptr<hittable> b) {
            return box_compare(a, b, 1);
        }

        static bool box_z_compare(const shared_ptr<hittable> a, const shared_ptr<hittable> b) {
            return box_compare(a, b, 2);
        }
    };
    ```

## Another BVH Optimization

Rather than choosing a random axis for each split, we can do better by always splitting along the *longest* axis of the bounding box covering the current object span. This tends to produce more balanced trees.

First we add `longest_axis()` to `Aabb`, along with `empty` and `universe` sentinel values:

<Tabs>
  <Tab label="Rust">
    ```rust filename="aabb.rs | longest_axis and sentinel constants"
    impl Aabb {
        pub fn longest_axis(&self) -> usize {
            if self.x.size() > self.y.size() {
                if self.x.size() > self.z.size() { 0 } else { 2 }
            } else {
                if self.y.size() > self.z.size() { 1 } else { 2 }
            }
        }

        pub const EMPTY: Aabb = Aabb {
            x: Interval::EMPTY,
            y: Interval::EMPTY,
            z: Interval::EMPTY,
        };

        pub const UNIVERSE: Aabb = Aabb {
            x: Interval::UNIVERSE,
            y: Interval::UNIVERSE,
            z: Interval::UNIVERSE,
        };
    }
    ```
  </Tab>

  <Tab label="C++">
    ```cpp filename="aabb.h | New aabb constants and longest_axis() function"
    class aabb {
      public:
        // ...

        int longest_axis() const {
            if (x.size() > y.size())
                return x.size() > z.size() ? 0 : 2;
            else
                return y.size() > z.size() ? 1 : 2;
        }

        static const aabb empty, universe;
    };

    const aabb aabb::empty    = aabb(interval::empty,    interval::empty,    interval::empty);
    ```
